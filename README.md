# 🩸 Sickle Cell Detection using Deep Learning

## 📌 Project Overview
This project detects Sickle Cell Anemia from blood cell images using deep learning models. 
It classifies images into:
- **Positives** → Sickle Cell Detected
- **Negatives** → Normal Cell

---

## 🧠 Models Used
- Simple CNN  
- Deep CNN  
- MobileNetV2 (Transfer Learning - Best Performance)

---

## 📂 Dataset
- Source: Kaggle (Sickle Cell Dataset)
- Dataset split into:
  - Train
  - Validation
  - Test

> Note: Dataset is not uploaded due to size limitations.
